Sleep Calculator

This Sleep Calculator is a Java program created using NetBeans and the Swing library. The program provides a simple graphical interface where users can enter their birth date and today’s date. When the Calculate button is clicked, it computes the total number of days the user has lived and estimates the total hours of sleep, assuming an average of seven hours of sleep per day. The calculations are based on basic arithmetic, converting years to days and approximating months as 30 days each. The results are then displayed in text fields on the interface. This project demonstrates the use of Java Swing components, event handling, and GUI design with NetBeans’ Form Editor to create an interactive application.

Digits of Pi (link to GitHub where all my project phases are available)

Digital Electronics (CP-220) Project Phases

For my CP-220 course, I designed a combinational logic circuit that outputs the first eleven digits of π based on a 4-bit binary input. Using AND, OR, and NOT gates, the circuit maps inputs 0–10 to the correct decimal digit and handles invalid inputs with outputs set to 0. I created truth tables, minimized logic equations using Karnaugh maps, and implemented the circuit with LEDs and optional 7-segment displays. The design was verified through simulation and hardware testing to ensure accurate digit outputs. This project demonstrates practical application of digital logic, Boolean algebra, and combinational circuit design.
